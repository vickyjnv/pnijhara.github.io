About Me

July 22, 2019

I am Prajjwal Nijhara (~ PNIJHARA). I am currently studying Electrical Engineering at Zakir Husain College of Engineering and Technology Aligarh Muslim University.

Though I love to try new technologies however I love to work on python programming. I love to work on team projects but when it comes with personal coding I try to learn new stuff, anywhere from creating API to trying new frameworks to reading about new tech.

Other than trying new pieces of stuff I am learning to contribute in Open source thus tried my luck in Gsoc’18 but couldn’t make it up to there.

I along with my college folks, had the privilege to code for the team “Narcodes”, and finished as the first runner up in the Smart India Hackathon 2019 (software edition) held in Guwahati, India where we worked on the project which is hosted here for Ministry of Statistics and PI.

Recently, I got selected as a computer team member for AUV ZHCET club which deals with Autonomous under-water vehicles, there I came to know about Robotics Operating System(ROS) thus started learning more about it.

Plans for the future

I am planning to switch to JavaScript for the web soon; owing to its industrial approach in getting things done. I’ve done things from application backend development to creating APIs for my projects, and creating responsive UIs from scratch using media queries. Along with this I will continue to explore more about python and might learn about machine learning.
